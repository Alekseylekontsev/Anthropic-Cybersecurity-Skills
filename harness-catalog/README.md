# Harness — AI Dev Factory: каталог кейсов Cybersecurity Skills

Каталог экспертных кейсов для **Claude Code Cybersecurity Skills**, оформленных в git-совместимом формате согласно иерархии Harness (АИ-Дев-Фабрики).

Каждый кейс показывает, как готовый к применению экспертный контекст (skill) удерживает AI-агента от "среза угла" — технически правдоподобного, но фактически опасного решения, которое без доменной экспертизы прошло бы ревью незамеченным.

## Иерархия

```
harness-catalog/
├── catalog.yaml                      # корневой манифест каталога
├── README.md
└── domains/
    └── <domain>/                     # напр. cybersecurity
        └── <subdomain>/              # напр. endpoint-security
            └── <case-id>/
                ├── case.md           # кейс в человекочитаемом формате (RU)
                └── skill.yaml        # структурированный манифест кейса
```

## Формат кейса (`case.md`)

Каждый кейс состоит из четырёх разделов:

1. **О skill** — что за упакованная методология передаётся агенту.
2. **Что произошло** — какой путь наименьшего сопротивления выбрал агент при валидации.
3. **Почему это важно** — почему решение проходит ревью и к чему приводит.
4. **Ценность скилла** — как скилл структурирует задачу и делает ревью предсказуемым.

## Источник

Кейсы построены на реальных скиллах из библиотеки Cybersecurity Skills (см. поле `source_skill` в каждом `skill.yaml`).
