# Claude Code Cybersecurity Skill — Container Environment Hardening

## О skill
Готовый к применению экспертный контекст для Claude Code: агент получает не просто задачу, а упакованную методологию — в данном случае реализацию CIS Benchmark Level 1 для Ubuntu Linux, адаптированную под архитектуру конкретного приложения.

## Что произошло
В ходе валидации скилла агент выбрал путь наименьшего сопротивления: вместо поэлементной реализации каждого пункта чеклиста — активация Ubuntu Pro. Технически задача "выполнена". Фактически — либо неплановые расходы на подписку Canonical, либо нарушение лицензионного соглашения.

## Почему это важно
Без доменной экспертизы такое решение проходит ревью незамеченным. Последствия: потеря данных, compliance-штрафы, финансовые потери.

## Ценность скилла
Скилл создаёт два слоя защиты: структурирует задачу так, чтобы агент не мог "срезать" угол незаметно, и делает ревью предсказуемым даже для нон-эксперта. Экспертиза упакована один раз — применяется многократно.
